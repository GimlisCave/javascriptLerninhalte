Finanz-Rechner

Funktionen bündeln wiederverwendbaren Code und nehmen Parameter entgegen.
Hier baust du verschiedene Finanz-Tools mit eigenen Funktionen.

📋
Kontoauszug (Konsole)
Funktion mit Parametern · Ausgabe per console.log()

Erstelle eine Funktion kontoauszug(name, kontonr, saldo), die einen formatierten Kontoauszug in der Konsole ausgibt. Öffne die Browser-Konsole (F12) um die Ausgabe zu sehen.

💸
Trinkgeld-Rechner
Funktion mit return · Ergebnis weiterverarbeiten

Schreibe eine Funktion trinkgeld(betrag, prozent), die das Trinkgeld berechnet und zurückgibt. Gib Trinkgeld und Gesamtbetrag im Div aus.

📈
Zinsrechner
Standardparameter · Fehlende Werte automatisch ersetzen

Erstelle eine Funktion zinsen(kapital, zinssatz = 3, jahre = 1). Wird kein Zinssatz oder keine Laufzeit angegeben, werden Standardwerte (3% / 1 Jahr) verwendet.

💱
Währungsrechner
onclick + Funktion · Euro in andere Währungen umrechnen

Gib einen Betrag in Euro ein. Per Klick auf den jeweiligen Button wird in die gewünschte Währung umgerechnet und das Ergebnis in #out-waehrung ausgegeben.

🎓
Kreditwürdigkeit
Funktion + If-Else · Score berechnen und bewerten

Berechne einen einfachen Kredit-Score. Eingabe: monatliches Einkommen und monatliche Ausgaben. Ausgabe: Sparquote in % und Bewertung (ausgezeichnet / gut / knapp / kritisch).
