Funktionen

neuer Input:







Aufgabe "Volumsberechnung": Erstellen Sie eine Funktion, die das Volumen berechnet. Sie können sich an den Übungen oben orientieren. Für das Volumen sind 3 Parameter nötig. Diesmal soll das Ergebnis in der Console ( mit console.log() ) ausgegeben werden.







Aufgabe "MwSt.-Rechner": Erstellen Sie eine Funktion, die die Mehrwertsteuer (20%) eines Betrages berechnet und im Div #mwst ausgibt.







neuer Input:







Eigene Aufgabe "Oberflächenberechnung": Erstellen Sie eine Funktion, die die Oberfläche eines Quaders berechnet. Sie können sich an der Übung oben orientieren. Für die Oberfläche sind 3 Parameter nötig. Wird ein Parameter nicht angegeben, soll standardmäßig der Wert 1 verwendet werden. Fügen Sie das Ergebnis im Div #oberflaeche ein







neuer Input:







Aufgabe "Millimeter in Kilometer": Dies ist ein Umrechner von Millimeter in Kilometer. Dabei soll der Besucher in ein Textfeld die Millimeter eingeben, welche er (mit Klick auf einen Button) in Kilometer umrechnen lassen möchte. Die Lösung soll im Div #kilometer erscheinen.







Aufgabe "Note berechnen": Es soll die maximal erreichbaren und die erreichten Punkte eingegeben werden. Ausgegeben wird die entsprechende Note und die erreichten Prozent. Den Notenschlüssel bestimmen Sie. Schreiben Sie die Lösung in den DIV #note hinein.










eigene Aufgabe "Umrechner C-F": Erstellen Sie einen Umrechner von Celsius in Fahrenheit. Nutzen Sie ein Eingabefeld. Sobald man auf den Button "umrechnen" klickt, wird die Lösung berechnet. Die Umrechnung folgt diesen Regeln: Zuerst die Celsius mit 1,8 (Achtung, man schreibt 1.8) multiplizieren, anschließend 32 addieren. Schreiben Sie die errechneten Fahrenheit in den Div #fahrenheit hinein.
