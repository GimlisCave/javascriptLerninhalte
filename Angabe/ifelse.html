If-Else-Abfrage

neuer Input:







Aufgabe "Monat": Lesen Sie das aktuelle Monat aus.
0 = Jänner, 11 = Dezember
Schreiben Sie in den DIV #monat das aktuelle Monat als Wort.






Aufgabe "Winterreifen": Geben Sie mittels einer Variable an, welches Monat gerade ist. Von November (11) bis April (4) soll "Winterreifen montieren" ausgegeben werden.
Erweitern Sie die Aufgabe mit der Abfrage, ob winterliche Fahrverhältnisse herrschen. Erst, wenn das Monat ODER die winterlichen Fahrverhältnisse zutreffen, soll die Info "Winterreifen montierren" ausgegeben werden.





Aufgabe "Bus": Ein Schulausflug steht an. Ihr JavaScript soll die Anzahl der SchülerInnen und Lehrer abfragen (Eingabe mit parseInt() in eine Zahl umwandeln). Welcher Bus wird vorgeschlagen? 50er, 40er, 20er oder ein Auto?
Schreiben Sie die Lösung in den DIV #bus hinein.






Aufgabe "Akku": Fragen Sie durch eine Eingabe die Akkuladung ab (Wert, den der Nutzer eingeben kann, ist zwischen 0 und 100).
Wenn Akkuladung: 0 - 25: Geben Sie "ROT! Akku schwach." aus.
Wenn Akkuladung: über 25: Geben Sie "Grün! Akku ok." aus.






Eigene Aufgabe "Begrüßung": Ihre Datei soll Sie im DIV #begruessung mit einem Text begrüßen. Dieser Text ist abhängig von der Tageszeit, welche mit new Date().getHours() abgefragt wird. Ist es noch vor 11 Uhr, so soll "Guten Morgen" ausgegeben werden, zwischen 11 und 16 Uhr, soll "Guten Tag" ausgegeben werden, danach soll "Guten Abend" zu lesen sein. Sie können sich am oberen Beispiel mit der Monatsausgabe orientieren.
