Aufgabe "Textauflistung": Es soll ein Fenster erscheinen, bei dem abgefragt wird, wie oft ein vordefinierter Text untereinander geschrieben werden soll. Anschließend soll der Text in korrekter Anzahl im folgenden Div erscheinen.






Aufgabe "Zahlenreihe": Schreiben Sie einen JavaScript-Code, der die Zahlen von 1-50 untereinander schreibt. Ausgabe im Div #zahlenreihe.






Eigene Aufgabe "Klassenbezeichnung": Es wird abgefragt, wie viele HLW-Klassen es gibt Anschließend sollen untereinander die HLW-Klassenbezeichnungen aufgelistet werden. Ausgabe im Div #zahlenreihe.
Bsp. bei Eingabe von 5:
1. HLW
2. HLW
3. HLW
4. HLW
5. HLW






Aufgabe "Aukku2": Es wird abgefragt, wie hoch Ihr Akkustand ist (Eingabe zwischen 0 und 100). Abhängig davon wird die Akkuladung visuell ausgegeben. Ausgabe im Div #akku2.

Bsp.:
100% Akku = XXXXXXXXXX
50% Akku = XXXXX00000
51% Akku = XXXXXX0000
3% Akku = X000000000






Aufgabe "halbe Pyramide": Schreiben Sie einen JavaScript-Code, der folgende 20-zeilige halbe Pyramide mit dem Buchstaben "x" und einem darauffolgenden Leerzeichen erzeugt. Als Schriftart wählen Sie "Courier New".







SCHWIERIGE Aufgabe "ganze Pyramide": Schreiben Sie einen JavaScript-Code, der folgende ganze Pyramide erstellt. Der Benuzter soll in einem Input-Feld die Anzahl der Zeilen und das gewünschte Zeichen (statt dem X) eingeben. Als Schriftart wählen Sie "Courier New". Um die Zeichen mittig einrücken zu können, benötigen Sie viele Leerzeichen, welche mit   erzeugt werden können.
